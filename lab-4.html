Part 2 - Misrepresentation

Question 4

The following chart was shared by @GraphCrimes on X/Twitter on September 3, 2022.

  1. What is misleading about this graph?

The graph is misleading because it visually exaggerates the dominance of the public sector responses. Although the percentages (e.g., 94% vs 3%) are accurate, the bar design makes the public sector portion appear overwhelmingly larger than it actually is.

Additionally, all categories are stacked in a single bar, which makes it difficult to directly compare the smaller categories (private sector and “don’t know”) across services. The visual emphasis strongly draws attention to the public sector while minimizing the visibility of other responses.

  1. Suppose you wanted to recreate this plot, with improvements to avoid its misleading pitfalls from part (a). You would obviously need the data from the survey in order to be able to do that. How many observations would this data have? How many variables (at least) should it have, and what should those variables be?

The data would have 1858 observations, since the chart states that the total number of respondents is 1858.

At minimum, the dataset should have two variables:

  • service: the type of service (Rail, Water, Energy, Royal Mail)
  • response: the respondent’s answer (Public sector, Private sector, Don’t know)

Each row would represent one respondent’s answer for one service. Alternatively, the dataset could also include a third variable such as count or percentage if the data are aggregated.

survation_long <- survation %>%
  pivot_longer(
    cols = c(`Royal Mail`, Energy, Water, Rail),
    names_to = "service",
    values_to = "response"
  )

survation_summary <- survation_long %>%
  group_by(service, response) %>%
  summarise(n = n(), .groups = "drop") %>%
  group_by(service) %>%
  mutate(percent = n / sum(n) * 100)

survation_summary$service <- factor(
  survation_summary$service,
  levels = c("Rail", "Water", "Energy", "Royal Mail")
)

survation_summary$response <- factor(
  survation_summary$response,
  levels = c("Public sector", "Private sector", "Don't know")
)

ggplot(survation_summary, aes(x = percent, y = service, fill = response)) +
  geom_col() +
  scale_fill_manual(
    values = c("Public sector" = "#006697",
               "Private sector" = "#FF3205",
               "Don't know" = "gray")
  ) +
  labs(
    title = "Q4. Do you think the following services should be run in the private\nsector or the public sector?",
    caption = "Base: All Respondents Unweighted Total: Total=1858",
    x = NULL,
    y = NULL,
    fill = NULL
  ) +
  theme_minimal() +
  theme(
    legend.position = "bottom"
  )

The improved visualization looks cleaner and less visually exaggerated than the original. It presents the same percentages, but it avoids drawing too much attention to the largest category through heavy styling. At first glance, the improved plot still shows that most respondents preferred the public sector, but it feels more balanced and easier to compare across all response categories.
